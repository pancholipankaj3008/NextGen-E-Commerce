# 🛍️ NextGen E-Commerce

A modern full-stack MERN clothing e-commerce platform with a responsive customer storefront and a powerful role-based admin dashboard.

---

## 📌 Overview

NextGen E-Commerce is a scalable online clothing store built using the MERN Stack. The project focuses on providing a complete shopping experience for customers while offering a powerful admin panel for managing products, inventory, orders, coupons, users, and analytics.

The application follows a modular architecture with REST APIs, JWT authentication, role-based authorization, and Cloudinary image management.

# ⚙️ Installation

Clone Repository

```bash
git clone https://github.com/your-username/NextGen-E-Commerce.git
```

Install Frontend

```bash
cd CLIENT
npm install
npm run dev
```

Install Backend

```bash
cd SERVER
npm install
node server.js
```

---

# 🔑 Environment Variables

Create a `.env` file inside the server directory.

```env
PORT=

MONGO_URL=

ACCESS=

REFRESH=

RESET=

BREVO_USER=

BREVO_PASS=

EMAIL_USER=

EMAIL_PASS=

CLOUDINARY_CLOUD_NAME=

CLOUDINARY_API_KEY=

CLOUDINARY_API_SECRET=
```
